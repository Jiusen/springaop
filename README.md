# springaop的学习

## SpringAOP使用详解 

### 1、面向切面编程
#### 编程范式概述
1.面向过程编程  
2.面向对象编程  
3.函数式编程  
4.事件驱动编程  
5.面向切面编程  

#### AOP是什么
1.是一种编程范式,不是编程语言  
2.解决特定问题，不能解决所有问题  
3.是OOP的补充，不是替代  

#### AOP的初衷
DRY: Don`t Repeat Yourself
Soc: Separation of Concerns
1. -水平分离: 展示层->服务层->持久层  
2. -垂直分离: 模块划分（订单、存库等）  
3. -切面分离: 分离功能性需求与非功能性需求  

#### 使用AOP的好处
1.集中处理某一关注点/横切逻辑  
2.可以很方便地添加/删除关注点  
3.侵入性少，增强代码可读性及可维护性  

#### AOP的应用场景
1.权限控制  
2.缓存控制  
3.事务控制  
4.审计日志  
5.性能监控  
6.分布式追踪  
7.异常处理  

### 切面表达式
#### Wildcards（通配符）
1. * 匹配任意数量的字符
2. + 匹配指定类及其子类
3. ..一般用于匹配任意数的子包或参数
